# FSDL Assignment

## Objective
To learn Git concepts including local and remote repositories, push, pull, branch, and merge.

## Steps Performed
1. Created a public GitHub repository
2. Cloned repository to local machine
3. Added files and committed changes
4. Created a new branch (feature1)
5. Added a new file in branch
6. Pushed branch to GitHub
7. Merged branch into main
8. Pushed final code to remote repository

## Commands Used
- git clone
- git add
- git commit
- git push
- git pull
- git branch
- git merge
